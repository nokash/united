Enterprice Resource Planning

We are working on a single ERP system, WFX-ERP-II. which unifies all the departments working. WFX Web ERP II - a new breed of Enterprise Resource Planning (ERP) software, which extends beyond a company's internal processes and connects to suppliers and buyers. It maintains the functionality of inward looking core ERP Apparel systems, and allows for features and functionality that can connect our entire value chain.
